  1. 게시글 하나를 열어서 선택 상태로 만듭니다.
  2. MongoDB vs Redis 읽기 비교를 눌러 평균 시간을 계산합니다.
  3. 선택 글 캐시 비우기를 누른 뒤 다시 측정해 차이를 봅니다.
