미니 Redis 웹 데모

게시판 + 인기글 + 로그인을 한 화면에서 보는 Redis 데모

버튼을 눌러보면 화면이 바뀌고, 동시에 개발자 콘솔에는 Redis 명령이 어떻게 실행되는지가 한글로 설명됩니다. 세션, 조회수, 캐시가 실제 웹페이지에서 어떻게 쓰이는지 직관적으로 볼 수 있습니다.

팁: F12를 눌러 Console 탭을 열어두세요. 로그인, 게시글 열람, 인기글 갱신을 할 때마다 요청, 응답, Redis 명령, 동작 이유가 모두 한글로 출력됩니다.

로그인 세션

로그인하면 session:<token> 키가 저장되고, 로그아웃하면 그 키가 삭제됩니다.

로그아웃 상태
현재 Redis에 활성 세션 키가 없습니다.

게시판 글 목록

게시글을 열면 INCR views:post:id 가 실행됩니다. 동시에 인기글 캐시를 지워서 순위를 다시 계산할 수 있게 만듭니다.

인기글 캐시

첫 번째 조회는 느린 계산을 흉내 내고, 그 다음 조회부터는 cache:top_posts 에서 바로 가져올 수 있습니다.

아직 요청 전

선택한 게시글

마지막으로 연 게시글과, 조회수 증가 결과가 이 패널에 바로 반영됩니다.

아직 연 게시글이 없습니다

아래 게시글의 "게시글 열기" 버튼을 누르면 조회수가 올라가고 인기글 캐시가 무효화됩니다.

첫 동작 이후 여기에 Redis 명령 요약이 한글로 표시됩니다.

현재 Redis 상태

지금 메모리에 들어 있는 키 목록입니다. 세션 키, 게시글 조회수 키, 인기글 캐시 키를 확인해보세요.
